# App Directory

## Overview
Contains the app's entry point and root views.

## Key Files
- `Service_CenterApp.swift`: App entry point and configuration
- `ContentView.swift`: Root view of the application

## Guidelines
1. Keep root views simple
2. Use dependency injection for services
3. Configure app-wide settings here
4. Coordinate changes with team

## Modifying App Entry Point
1. Create feature branch
2. Discuss changes with team
3. Update documentation
4. Create pull request

## State Management
- Document any app-wide state
- Use appropriate property wrappers
- Consider using dependency injection
